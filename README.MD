# Финансовый планер
## Описание проекта
### Настольное приложение Python для расчета доходов и расходов с анализом и визуализацией данных

## Функционал
- Добавление доходов или расходов (дата по умолчанию - текущая);
- Хранение данных - в CSV;
- Круговой график расходов и доходов по категориям;
- Предварительный просмотр данных;
- Построчное удаление данных;
- GUI Tk.
## Запуск
- pip install -r requirements.txt
- в проект добавить pandas и matplotlib (file - settings - python interpreter - нажать плюс)
- python main.app
